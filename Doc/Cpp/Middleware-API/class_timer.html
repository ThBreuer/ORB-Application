Ausführliche Beschreibung

Zeitmessung und Timeout.

Mit Objekte dieser Klasse können Zeitmessungen, Timeouts ud Wartezeiten realisiert werden.

  • Zeitmessung

    Die Zeitmessung liefert die Zeit, die seit dem Erzeugen des Objektes bzw. nach einem Reset des Objektes vergangen ist.

    Siehe auch
    get() und reset()

  • Timeout

    Nach dem Starten der Timeout-Funktion liefert Timer::timeout() einmalig true zurück, wenn die Laufzeit überschritten wurde. In diesem Falle wird der Timer zurückgesetzt und neu gestartet.

    Siehe auch
    start(), stop() und timeout()
  • Wartezeit

Alle Zeiten werden in ms (Millisekunden) angegeben
