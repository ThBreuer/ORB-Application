Ausführliche Beschreibung

Modellbau-Servo steuern.

Mit einem Objekt dieser Klasse läßt sich ein Modellbau-Servo steuern.
Der Winkel wird in Grad (Vollkreis = 360°) angegeben.

Warnung
Die Methoden sind noch nicht vollständig implementiert
